h1. Fake: Fixture generator plugin for cAKEphp.

This console plugin generate fixture using datadase data.

h2. Features

* Generate fixture using database data.

h2. Usage

@cake fake@
@cake fake all@

h2. When you use CakePHP 1.3

@cake bake fixture@

see "CookBook":http://book.cakephp.org/view/1611/Bake-improvements-in-1-3 .
